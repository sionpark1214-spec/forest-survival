# 🌲 숲에 생존 (Forest Survival)

3D 숲 생존 모바일 게임. 비행기가 추락했다. 살아남아라!

- ▶ 플레이: https://sionpark1214-spec.github.io/forest-survival/
- 단일 `index.html` — 브라우저로 열면 바로 실행 (Three.js 3D)

## 🎮 어떻게 살아남나
- 왼쪽 **조이스틱**으로 이동, 오른쪽 **공격/행동/제작/가방** 버튼
- 5가지 상태바: ❤️체력 · 🍖배고픔 · 💧목마름 · ❄️체온 · 💪힘(스태미나)
- 나무 베고 돌 캐서 **무기·집** 만들기 (나무창/돌도끼/활/모닥불)
- 🐔🐰🐍🐷🐺🐯 동물 사냥(다 식량!), 🐟물고기는 나무창으로, 🐛벌레는 손으로
- 날고기 먹으면 **독**! 모닥불에 구워먹기, 독은 🩹약초로 치료
- 깨끗한 물(강·샘물) 마시기, 더러운 물은 체력 -5
- 🌙밤·🌧️비엔 체온이 빨리 떨어짐
- 🏠 집 5등급 — 좋은 집일수록 침대 회복 빵빵
